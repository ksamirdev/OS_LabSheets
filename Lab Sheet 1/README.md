# Lab Sheet 1 - ENCS351 Operating System

## Overview
This project implements process creation and management operations using Python's OS module, simulating Linux process management behaviors. The lab covers process creation, command execution, process states (zombie/orphan) and process inspection.

## Requirements

### System Requirements
- **Operating System**: Linux, macOS, or Windows with WSL
- **Python**: Version 3.6 or higher
- **Terminal/Command Line**: Basic command line access

### Python Dependencies
The project uses only Python standard library modules:
- `os`
- `time` 

No additional packages need to be installed.

### How to Run

```bash
python3 process_management.py
```
